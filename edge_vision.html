EdgeVision is designed to enhance real-time object detection and tracking capabilities at the edge using the Raspberry Pi 4 and its high-resolution Camera Module V2. Leveraging the Faster R-CNN ResNet-50 deep learning model, EdgeVision achieves accurate, real-time object counting. It is highly suitable for various applications such as public safety monitoring, traffic management, and retail analytics.

The system incorporates Centroid Tracking to identify and track objects reliably across multiple video frames. This ensures robust and consistent analytics, even under challenging conditions. Data from object counting operations can be stored in local or cloud-based SQL databases, enabling comprehensive analysis and historical pattern tracking. Future deployments plan to utilize cloud platforms like Heroku, allowing users convenient and device-independent access to analytics results.

A key highlight of EdgeVision is its user-friendly, interactive interface. Users can draw line segments onto live video streams, acting as virtual sensors to count objects passing through automatically.
